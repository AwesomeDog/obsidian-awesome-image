## 关于插件
当你点击一个图片时，该图片将被会弹出，此时您可以预览查看、拖拽、缩放、旋转图片。

![view_image.png](example/view_image.gif)

* 同时，还可以通过鼠标滑轮实现图片的放大和缩小，通过 Esc 键来实现图片预览弹出层的关闭。
